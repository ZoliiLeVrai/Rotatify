Comment utiliser un proxy ?

Configuration sur Firefox

Pour naviguer sur Internet via nos proxy, nous vous conseillons d'utiliser Firefox. Voici comment le configurer.

Cliquez sur l'icon en haut a droite de Firefox puis paramètres. Tapez proxy dans le champ de recherche en haut a droite pour faire apparetre le bonton paramètres.. réseau. Cochez la case Configuration manuelle du proxy et Utiliser également ce proxy pour HTTPS, puis entrez l'ip et le port de votre proxy dans Proxy HTTP et HTTPS.

Vous pouvez commencer à naviguer sur Internet, un pop-up apparaîtra pour vous demander d'entrer le nom d'utilisateur et le mot de passe du proxy.
